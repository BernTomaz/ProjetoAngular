Data Binding

Data binding é uma técnica que permite a sincronização entre o modelo e a visão.

O Angular oferece quatro tipos principais de data binding:


  • Interpolation: Usado para exibir dados do componente na visão.

    O número de cliques é: {{ contadorClique }}


  • Property Binding: Usado para definir propriedades de elementos HTML.





  • Event Binding: Usado para capturar eventos do usuário.


  • Two-Way Data Binding: Permite a comunicação bidirecional entre o modelo (TS) e a visão (HTML).
